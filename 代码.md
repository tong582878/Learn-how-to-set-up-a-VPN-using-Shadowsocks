# 推荐使用ubuntu22及以上系统，服务器1核1G就可以了

## 1. 系统更新
```bash
sudo apt update && sudo apt upgrade -y
```
## 2.安装Shadowsocks-libev
```bash
sudo apt install shadowsocks-libev -y
```
## 3.修改config配置
```bash
sudo vim /etc/shadowsocks-libev/config.json
```
```json
{
    "server": ["::", "0.0.0.0"],
    "server_port": 8888,
    "password": "3e7b635f-a047-4cf8-8fa0-0d51017089e3", 
    "method": "chacha20-ietf-poly1305", 
    "mode": "tcp_and_udp",
    "fast_open": true,
    "no_delay": true,
    "reuse_port": true,
    "nameserver": "1.1.1.1"
}
```
## 4.网络参数优化
```bash
sudo vim /etc/sysctl.d/local.conf
```
```conf
fs.file-max = 51200 
net.core.rmem_max = 67108864 
net.core.wmem_max = 67108864 
net.core.netdev_max_backlog = 4096 
net.core.somaxconn = 4096 
net.ipv4.tcp_syncookies = 1 
net.ipv4.tcp_tw_reuse = 1 
net.ipv4.tcp_fin_timeout = 30 
net.ipv4.tcp_keepalive_time = 1200 
net.ipv4.ip_local_port_range = 10000 65000 
net.ipv4.tcp_max_syn_backlog = 4096 
net.ipv4.tcp_max_tw_buckets = 5000 
net.ipv4.tcp_fastopen = 3 
net.ipv4.tcp_rmem = 4096 87380 67108864 
net.ipv4.tcp_wmem = 4096 65536 67108864 
net.ipv4.tcp_mtu_probing = 1 
net.ipv4.tcp_congestion_control = bbr
```
```bash
sudo sysctl --system
```
## 5.启用bbr
```bash
sudo modprobe tcp_bbr echo "tcp_bbr" | sudo tee -a /etc/modules-load.d/modules.conf echo "net.core.default_qdisc=fq" | sudo tee -a /etc/sysctl.conf 
echo "net.ipv4.tcp_congestion_control=bbr" | sudo tee -a /etc/sysctl.conf 
sudo sysctl -p
```
## 6.配置UFW防火墙
```bash
sudo ufw allow ssh
sudo ufw allow 8888/tcp
sudo ufw allow 8888/udp
sudo ufw enable
```
## 7.启动并检查服务
```bash
sudo systemctl start shadowsocks-libev
sudo systemctl status shadowsocks-libev
```

## 8.客户端连接
服务器IP：你自己的公网ip  
端口：8888  这个自己改，上面的config文件和ufw的端口也要同步修改
密码：3e7b635f-a047-4cf8-8fa0-0d51017089e3 用自己的，搞复杂一点
加密方式：chacha20-ietf-poly1305 这个加密方式建议就这个，或者你根据需要自己在上面的config文件去修改
